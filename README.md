# air-quality-monitor

Para la ejecución correcta de la implementación en Streamlit que se realizó, se debe hacer lo siguiente:

- Descargar y descomprimir el proyecto.
- Ubicarse donde está el dashboard.py y sanitize_data.py (por default en workshop1)
- Abrir un cmd en el directorio donde se encuentran.
- Ejecutar primero el comando: streamlit run sanitize_data.py (Para tener los archivos .json formateados correctamente)
- Luego, se ejecuta: streamlit run dashboard.py

Allí se abrirá una página en localhost y podrá ver lo que se realizó.
Nota: Dentro del archivo 'sensor_data.json' se encuentran los 10.000 .json que se usaron de prueba.
